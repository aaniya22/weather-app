# Weather

A clean, editorial weather app built with React and the OpenWeatherMap API.

## Features

- Real-time weather data for any city
- Temperature, condition, humidity, wind speed, feels like, high/low
- Background tint shifts based on weather condition
- Skeleton loading state
- Mobile responsive

## Tech Stack

- React 18
- Vite
- OpenWeatherMap API
- Custom hooks
- CSS variables

## Run Locally

```bash
git clone https://github.com/aaniya22/weather-app.git
cd weather-app
npm install
```

Create a `.env` file in the root:

```
VITE_WEATHER_API_KEY=your_api_key_here
```

```bash
npm run dev
```
