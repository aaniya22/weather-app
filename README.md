# Weather App 🌤️

A responsive weather application built with React and the OpenWeather API.

## Features
- Search any city worldwide
- Displays real-time temperature, humidity, wind speed, and weather conditions
- Dynamic UI updates based on weather data
- Error handling for invalid city names

## Tech Stack
- React.js
- OpenWeather API
- CSS

## Live Demo
[View Live](https://weather-app-nu-puce-39.vercel.app)

## Getting Started
```bash
git clone https://github.com/aaniya22/weather-app.git
cd weather-app
npm install
npm run dev
```
